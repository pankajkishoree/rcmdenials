📄 Documentation Requirements

  • ✓ Comprehensive eye exam: visual acuity, intraocular pressure, dilated fundus examination
  • ✓ Cataract surgery: pre-operative assessment, IOL calculation, and post-operative outcome
  • ✓ Retinal injection: indication, agent injected, dose, lot number, and injection technique
  • ✓ Glaucoma documentation: IOP, cup-to-disc ratio, visual field results, and OCT findings
  • ✓ OCT results with measurements and comparison to prior studies
  • ✓ Visual field test results with reliability indices and comparison to prior fields
  • ✓ Diabetic retinopathy grading with ETDRS classification when applicable
  • ✓ Fundus photography documentation with lesion description and location
  • ✓ Laterality documentation for every unilateral procedure (RT/LT)
  • ✓ Prior treatment history and failure documentation for repeat injections

⏱️ Global Surgery Period

Global Period Rules:

0 days for office-based eye exams, OCT, visual fields, and retinal injections. 10 days for minor eye procedures (e.g., chalazion removal, punctal dilation). 90 days for major procedures (cataract surgery, vitrectomy, glaucoma surgery). Post-operative visits included in global surgical package.

💡 Medical Necessity Tips

  • ✓ Document visual acuity measurements for every eye exam and procedure
  • ✓ Include intraocular pressure and cup-to-disc ratio for glaucoma patients
  • ✓ Document OCT measurements and comparison to prior studies for retinal disease
  • ✓ Reference AAO/AAO guidelines for appropriate use criteria in ophthalmology procedures
  • ✓ Document functional visual impairment affecting daily activities for cataract surgery
  • ✓ Include fundus photography and OCT results for diabetic retinopathy grading
  • ✓ Document prior injection history and interval changes for repeat anti-VEGF therapy
  • ✓ Ensure laterality (RT/LT) documented for every unilateral procedure

📝 Billing Guidelines

  • ✓ Always include RT or LT modifier for unilateral eye procedures — laterality required
  • ✓ Use modifier 50 for bilateral same-day cataract surgery or bilateral injections
  • ✓ Append modifier 25 to E/M when billing on same day as eye procedure
  • ✓ Bill cataract surgery with single code (66984 or 66821) — do not bill IOL separately
  • ✓ Ensure anti-VEGF injection is billed with appropriate agent code (J-codes for drug)
  • ✓ Bill each eye separately if bilateral procedures performed on different dates
  • ✓ Use appropriate place of service codes — POS 11 (office), POS 22 (outpatient hospital)
  • ✓ Ensure OCT and visual field tests are billed with professional component (modifier 26) if reading only

📊 Example Claim Scenarios

Cataract Surgery with IOL Implant

CPT + ICD-10: 66984 (Extracapsular cataract removal with IOL) + H25.9 (Age-related cataract)

Modifiers: -RT (right eye) or -LT (left eye)

Notes: Patient with visually significant cataract, BCVA 20/70. Functional limitation documented. IOL implanted. Post-op visual acuity 20/25. Documentation includes pre-op assessment, IOL calculation, and post-op outcome.

Anti-VEGF Injection for Wet AMD

CPT + ICD-10: 67028 (Intravitreal injection) + H35.32 (Exudative AMD) + J-codes for drug

Modifiers: -RT or -LT (laterality required)

Notes: Patient with wet AMD, declining visual acuity. OCT shows new subretinal fluid. Anti-VEGF injection performed. Documentation includes visual acuity, OCT measurements, and injection technique. Drug billed separately with J-codes.

Glaucoma Evaluation with OCT and Visual Field

CPT + ICD-10: 92014 (Comprehensive eye exam) + 92134 (OCT macula) + 92081 (Visual field) + H40.11x0 (POAG)

Modifiers: -RT or -LT for unilateral testing

Notes: Annual glaucoma evaluation. IOP measured, cup-to-disc ratio documented. OCT shows RNFL thinning. Visual field shows nasal defect. All results compared to prior studies.
