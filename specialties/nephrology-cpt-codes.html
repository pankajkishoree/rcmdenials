📋 Documentation Requirements

  • ? Dialysis records including session duration, ultrafiltration volume, and pre/post weights
  • ? Vascular access site documentation  location, type, patency status, and complications
  • ? Lab values documented  BUN, creatinine, GFR, potassium, phosphate, calcium, albumin
  • ? Transplant evaluation records  immunologic testing, crossmatch results, surgical evaluation
  • ? Vascular access imaging reports  ultrasound mapping, fistulogram results
  • ? CKD staging documentation  confirmed GFR measurements over 3 months
  • ? Physician orders for dialysis type and frequency with clinical justification
  • ? Access function assessment  thrill, bruit, flow rates documented each session
  • ? Renal biopsy pathology report with indication and histological findings
  • ? Kt/V and urea reduction ratio (URR) for dialysis adequacy documentation

⏱️ Global Surgery Period

Global Period Rules:

0 days for dialysis sessions (90935-90966)  these are not surgical procedures. 90 days for kidney transplant surgery (50300, 50320, 50340, 50360, 50365). 0 days for vascular access procedures (36145, 36146, 36818, 36819, 36821) as they are typically considered minor outpatient procedures with a 0-day global. 0 days for renal biopsy (50547) and imaging studies.

✅ Medical Necessity Tips

  • ? Always document CKD staging with confirmed GFR values  use N18.31 (3a) vs N18.32 (3b) specificity
  • ? Include current lab values (BUN, creatinine, GFR, potassium) in clinical notes for dialysis justification
  • ? Document dialysis frequency and adequacy metrics (Kt/V, URR) to support ongoing treatment
  • ? For transplant evaluation, document why patient is a candidate  age, comorbidities, access type
  • ? Reference KDIGO guidelines for CKD management and dialysis initiation thresholds
  • ? Document vascular access complications (stenosis, thrombosis, infection) with clinical findings
  • ? For ESRD monthly codes, document visit frequency and complexity  do not upcode visits
  • ? Ensure ICD-10 specificity  E11.22 vs E10.22, I12.0 vs I12.9 for secondary causes of CKD

💰 Billing Guidelines

  • ? Correct ESRD coding  bill 90963 (initial month) then 90964-90966 (subsequent months) based on visits
  • ? Dialysis codes include E/M  do not bill separate E/M without modifier 25 documentation
  • ? Access management  bill 36145 for first access site, 36146 for each additional site separately
  • ? AV fistula  select 36818 (autogenous) vs 36819 (nonautogenous) based on graft type
  • ? Use modifier 59 for separately identifiable procedures performed same day as dialysis
  • ? Bill home dialysis management codes (90955-90958) separately from facility-based dialysis
  • ? Monitor NCCI edits  dialysis codes have specific bundling rules with access procedures
  • ? For imaging, bill 26/TC split in hospital outpatient setting  nephrologist bills professional component

📝 Example Claim Scenarios

Initiation of Hemodialysis

CPT + ICD-10: 90935 (HD with E/M, 1st hour) + N18.6 (ESRD) + Z99.2 (Dependence on renal dialysis)

Modifiers: -25 (if separate E/M performed same day)

Notes: Patient with ESRD initiating hemodialysis. First session documented with pre/post weights, ultrafiltration volume, and lab values. Dialysis prescription and clinical justification in chart.

AV Fistula Creation

CPT + ICD-10: 36818 (AV anastomosis, autogenous) + N18.5 (CKD stage 5) + I71.2 (Aortic aneurysm, not documented)

Modifiers: -RT or -LT for laterality

Notes: Nephrologist creates left radiocephalic AV fistula for dialysis access. Vascular access ultrasound mapping performed prior. Documentation includes vessel size assessment and surgical rationale.

Kidney Transplant Evaluation

CPT + ICD-10: E/M 99215 (Complex E/M) + N18.6 (ESRD) + Z99.2 (Dependence on renal dialysis)

Modifiers: -25 for E/M if separate from dialysis session

Notes: Comprehensive transplant evaluation including immunologic workup, surgical assessment, and psychosocial evaluation. Documentation includes dialysis history, comorbidities, and transplant candidacy determination.
