📄 Documentation Requirements

  • ✓ Session start and stop times documented to verify code time threshold met
  • ✓ Treatment plan updated at least annually with current diagnoses and goals
  • ✓ Progress toward treatment goals documented with measurable outcomes
  • ✓ Risk assessment documented at each visit (suicidal ideation, self-harm, harm to others)
  • ✓ Medical decision-making complexity documented for E/M level selection
  • ✓ For medication management, document rationale for current medications and adjustments
  • ✓ Telehealth services documented with modality (audio-video or audio-only) and patient location
  • ✓ Informed consent for treatment documented and signed
  • ✓ For E/M plus psychotherapy add-on, document separately identifiable services
  • ✓ Neuropsychological testing reports with scores, interpretation, and clinical recommendations

⏱️ Global Surgery Period

Global Period Rules:

Psychiatry services have no global surgery period. Psychotherapy, evaluations, and medication management are standalone services not subject to surgical global periods. ECT has a 0-day global period. TMS has a 0-day global period for standard treatment courses.

💡 Medical Necessity Tips

  • ✓ Always document specific DSM-5 diagnoses with supporting clinical evidence
  • ✓ Document functional impairment level and impact on daily activities and work performance
  • ✓ Include treatment modality rationale — why this type of therapy for this diagnosis
  • ✓ Document treatment response and progress toward measurable goals at each session
  • ✓ For medication management, document current regimen, side effects, and adjustment rationale
  • ✓ Document frequency and duration justification for sessions exceeding weekly visits
  • ✓ For testing, document specific clinical questions being addressed by the evaluation
  • ✓ Include collateral information sources when applicable (family, school, other providers)

📝 Billing Guidelines

  • ✓ Bill psychotherapy code based on actual face-to-face time — document start/stop times
  • ✓ Use add-on codes (90833/90836) when E/M and psychotherapy are separately identifiable same day
  • ✓ Do not bill 90837 when session is under 53 minutes — use 90834 for 38-52 minutes
  • ✓ For new patient psychiatric eval, use 90791 (no medical services) or 90792 (with medical services)
  • ✓ Bill screening codes (96127) separately only when scoring and interpretation are documented
  • ✓ Use modifier 95 for telehealth visits provided via audio-video connection
  • ✓ For family psychotherapy, document specific family members involved and therapy content
  • ✓ Monitor visit frequency limits — document justification for more than weekly sessions

📊 Example Claim Scenarios

Initial Psychiatric Evaluation with Medication Management

CPT + ICD-10: 90792 (Psychiatric eval with medical services) + F32.1 (Major depressive disorder, moderate)

Modifiers: -95 (telehealth visit)

Notes: New patient presenting with depressive symptoms. Comprehensive evaluation includes history, mental status exam, diagnosis, and medication plan. Evaluation takes 60+ minutes. Documentation includes treatment rationale and follow-up plan.

Psychotherapy Session with E/M Add-On

CPT + ICD-10: 99214 (E/M established patient) + 90836 (Psychotherapy 45 min add-on) + F41.1 (Generalized anxiety disorder)

Modifiers: -25 (separately identifiable E/M), -95 (telehealth)

Notes: Established patient returns for medication management and therapy. E/M addresses medication adjustment and psychotherapy provides cognitive behavioral interventions. Both services separately documented with distinct notes.

Neuropsychological Testing for ADHD

CPT + ICD-10: 96132 (Neuropsych testing physician 30 min) + 96133 (add-on 30 min x3) + F90.0 (ADHD predominantly inattentive)

Modifiers: None required

Notes: Patient referred for attention difficulties. Two-hour neuropsychological evaluation includes cognitive testing, attention measures, and behavioral assessment. Documentation includes scores, interpretation, and diagnostic recommendations.
