📄 Documentation Requirements

  • ✓ Age-appropriate growth and development assessment documented at each well visit
  • ✓ Immunization records with vaccine name, manufacturer, lot number, and injection site
  • ✓ Anticipatory guidance topics documented (nutrition, safety, developmental milestones)
  • ✓ Screening tests performed (vision, hearing, lead, autism) with results documented
  • ✓ Physical examination findings including vital signs with percentiles for age/sex
  • ✓ Parent/guardian counseling provided and questions addressed documented
  • ✓ For sick visits, document HPI, ROS, physical exam, assessment, and plan
  • ✓ Behavioral health screening (PHQ-A for adolescents) documented when performed
  • ✓ Newborn screening results and follow-up documented
  • ✓ Sports physical documentation including cardiac and musculoskeletal screening

⏱️ Global Surgery Period

Global Period Rules:

0 days for most pediatric procedures. Immunization administration has no global period. Well-child visits are standalone preventive services not subject to surgical global periods.

💡 Medical Necessity Tips

  • ✓ Document age-appropriate developmental milestones and screening results at each visit
  • ✓ Include immunization due dates and catch-up schedules in medical records
  • ✓ Document parental counseling topics and any concerns discussed during well visits
  • ✓ For sick visits, document clinical rationale for antibiotic or treatment selection
  • ✓ Include growth chart documentation with BMI percentile and weight gain patterns
  • ✓ Document sports physical screening including cardiac history and physical findings
  • ✓ For chronic conditions (asthma, ADHD), document current control status and medication compliance
  • ✓ Include behavioral health screening results (PHQ-A, M-CHAT) with clinical interpretation

📝 Billing Guidelines

  • ✓ Bill well-child visit with correct age-based preventive code (99381-99385 new, 99391-99395 established)
  • ✓ Use modifier 25 when billing E/M on same day as well-child visit for separate acute illness
  • ✓ Bill vaccine administration codes (90460/90461) separately from vaccine procurement codes
  • ✓ Ensure correct place of service — office (11) vs outpatient hospital (22) affects reimbursement
  • ✓ Monitor preventive visit frequency limits — typically one well visit per 12-month period per age bracket
  • ✓ For telehealth visits, use modifier 95 and document audio-video modality and patient location
  • ✓ Bill school/sports physical forms completion with code 99081 when documented separately
  • ✓ Verify whether facility or professional component is being billed for diagnostic services

📊 Example Claim Scenarios

15-Month Well-Child Visit with Immunizations

CPT + ICD-10: 99392 (Well-child established 1-4 years) + 90460 (Vaccine admin first) + 90461 (Vaccine admin add-on x3) + Z00.129 (Routine child exam)

Modifiers: -HA (Preventive visit with counseling)

Notes: 15-month established patient for well-child visit. M-CHAT screening negative. Developmental milestones met. 4 vaccines administered with counseling. Documentation includes growth chart, screening results, and anticipatory guidance topics.

Sick Visit E/M on Same Day as Well Visit

CPT + ICD-10: 99392 (Well-child established) + 99214 (Sick visit E/M) + J06.9 (URI) + Z00.129 (Routine exam)

Modifiers: -25 (Separately identifiable E/M)

Notes: Established patient scheduled for well visit presents with acute URI symptoms. Well visit performed first, then separate E/M for illness. Distinct documentation for each service. E/M documents fever, cough, throat exam, and treatment plan.

New Patient Adolescent Visit

CPT + ICD-10: 99384 (New patient preventive 12-17 years) + 96160 (Caregiver health risk assessment) + Z00.121 (Child exam abnormal findings)

Modifiers: -HA

Notes: 14-year-old new patient for annual physical. PHQ-A screening positive for mild depression. Behavioral health concerns addressed. Documentation includes cardiac screening, scoliosis check, and mental health assessment with safety planning.
