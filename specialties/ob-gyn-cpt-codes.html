📄 Documentation Requirements

  • ✓ Maternity care: documented prenatal visits, gestational age, and pregnancy complications
  • ✓ Delivery documentation: method of delivery, indication for cesarean if applicable
  • ✓ Gynecologic procedure: indication, informed consent, findings, and post-op plan
  • ✓ Pap smear results with cytology interpretation for screening visits
  • ✓ Ultrasound documentation with measurements, gestational age, and fetal assessment
  • ✓ Biopsy documentation: site, method, and pathology ordered
  • ✓ Postpartum documentation: recovery status, contraception counseling, and follow-up
  • ✓ HPI, ROS, and physical exam findings for every OB/GYN E/M encounter
  • ✓ Maternal-fetal medicine consultation documentation when applicable
  • ✓ Informed consent documentation for all invasive procedures

⏱️ Global Surgery Period

Global Period Rules:

0 days for office-based gynecologic procedures (biopsy, Pap smear). 10 days for minor gynecologic surgery (e.g., cervical cone biopsy, dilation and curettage). 42-58 days for vaginal delivery. 90 days for cesarean delivery. The maternity global package (59400, 59510) includes all prenatal, delivery, and postpartum care within the global period.

💡 Medical Necessity Tips

  • ✓ Document gestational age and pregnancy dating for every prenatal visit
  • ✓ Include indication for cesarean delivery (failed induction, malpresentation, etc.)
  • ✓ Document abnormal Pap/HPV results for colposcopy and biopsy justification
  • ✓ Include uterine size, bleeding pattern, and symptom severity for gynecologic procedures
  • ✓ Reference ACOG guidelines for appropriate use criteria in obstetric care
  • ✓ Document fertility history and treatment failure before fertility intervention
  • ✓ Include informed consent and patient counseling documentation for all procedures
  • ✓ Document postpartum recovery status and contraception counseling

📝 Billing Guidelines

  • ✓ Bill maternity global package (59400/59510) — includes all routine prenatal, delivery, and postpartum
  • ✓ Use transfer codes (59610, 59622) when care transfers mid-pregnancy from another provider
  • ✓ Append modifier 25 to E/M when billing on same day as gynecologic procedure
  • ✓ Bill each component of maternity care separately when non-global billing selected
  • ✓ Ensure delivery code matches documented method (vaginal vs. cesarean)
  • ✓ Use appropriate trimester coding for pregnancy-related diagnoses
  • ✓ Bill ultrasound with fetal assessment codes (76805, 76811) when performed
  • ✓ Verify global period exclusions — complications billed separately from global package

📊 Example Claim Scenarios

Global Maternity Package - Vaginal Delivery

CPT + ICD-10: 59400 (Global maternity, vaginal delivery) + Z34.81 (Normal pregnancy) + O23.90 (Urinary tract infection in pregnancy)

Modifiers: None — global package includes all services

Notes: Patient followed throughout pregnancy with routine prenatal visits. Delivered vaginally at 39 weeks. Global package includes all prenatal, delivery, and 6-week postpartum visit. Complications billed separately.

Colposcopy with Cervical Biopsy

CPT + ICD-10: 57452 (Colposcopy) + 57500 (Cervical biopsy) + N87.0 (Cervical dysplasia)

Modifiers: -59 for separate procedure if needed

Notes: Abnormal Pap result prompted colposcopy. Cervical biopsies taken from three sites. Documentation includes speculum examination findings, acetic acid application, and biopsy site descriptions.

Gynecologic E/M with Problem Visit

CPT + ICD-10: 99214 (Established patient, moderate complexity) + N92.1 (Excessive menstruation) + N80.0 (Fibroids)

Modifiers: None — standalone E/M visit

Notes: Patient presents with heavy menstrual bleeding. Uterine fibroids identified on ultrasound. MDM supports moderate complexity. Treatment options discussed including medical and surgical management.
