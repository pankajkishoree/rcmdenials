📄 Documentation Requirements

  • ✓ EEG: indication (seizure, syncope, encephalopathy), findings, and interpretation
  • ✓ EMG/NCS: specific muscles tested, nerve conduction velocities, and clinical correlation
  • ✓ Neurologic examination findings with detailed mental status, cranial nerve, and motor/sensory documentation
  • ✓ Seizure documentation: type, frequency, triggers, and medication history
  • ✓ Headache documentation: frequency, severity, triggers, and prior treatment history
  • ✓ Neuroimaging results (CT, MRI) correlated with clinical findings
  • ✓ Medication trial documentation with doses and response for seizure or headache management
  • ✓ Functional assessment and disability documentation for chronic neurologic conditions
  • ✓ EMG needle insertion documentation with muscles tested and findings per muscle
  • ✓ NCS: temperature documentation, stimulation technique, and waveform measurements

⏱️ Global Surgery Period

Global Period Rules:

0 days for diagnostic EEG, EMG/NCS, and office-based neurology procedures. 10 days for minor neurosurgery procedures (e.g., nerve biopsy). 90 days for major neurosurgery (craniotomy, spinal fusion). E/M visits and neurodiagnostic testing have no global period.

💡 Medical Necessity Tips

  • ✓ Document specific seizure type, frequency, and duration for EEG justification
  • ✓ Include specific nerve/muscle symptoms (weakness, numbness, pain) for EMG/NCS
  • ✓ Document headache frequency, severity, and prior treatment failure for neuroimaging
  • ✓ Reference AAN/AES guidelines for appropriate use criteria in neurology procedures
  • ✓ Include medication trial history with doses and response for seizure or headache
  • ✓ Document functional impact on daily activities for disability evaluations
  • ✓ Include neurologic examination findings that support testing orders
  • ✓ Document prior neuroimaging results showing progression or new findings

📝 Billing Guidelines

  • ✓ Bill EEG with modifier 26 for professional component and TC for technical component when split
  • ✓ Use modifier 59 for separately identifiable EEG and EMG performed same day
  • ✓ Ensure correct NCS code selection based on number of nerves tested (95907-95913)
  • ✓ Document muscles tested for EMG to support code selection (95885, 95886)
  • ✓ Append modifier 25 to E/M when billing on same day as neurodiagnostic test
  • ✓ Bill each extremity separately for EMG/NCS with appropriate laterality modifiers
  • ✓ Use appropriate place of service codes — POS 11 (office), POS 22 (outpatient hospital)
  • ✓ Ensure EEG interpretation includes comparison with prior studies when available

📊 Example Claim Scenarios

EEG for Seizure Evaluation

CPT + ICD-10: 95816 (Routine EEG) + G40.909 (Epilepsy)

Modifiers: -26 for professional interpretation if billing physician only

Notes: Patient with two witnessed seizures. EEG ordered to evaluate for epilepsy. Documentation includes seizure description, frequency, and medication history. EEG shows epileptiform discharges.

EMG/NCS for Carpal Tunnel

CPT + ICD-10: 95910 (NCS, 3-4 studies) + 95885 (EMG, 4-5 muscles) + G56.00 (CTS)

Modifiers: -RT (right hand) or -LT (left hand)

Notes: Patient with numbness and tingling in median nerve distribution. NCS shows decreased conduction velocity. EMG shows denervation in median-innervated muscles. Clinical correlation with carpal tunnel syndrome.

Emergency Neurology Consult

CPT + ICD-10: 99285 (ED visit, comprehensive) + I63.9 (Cerebral infarction) or G45.9 (TIA)

Modifiers: None — standalone emergency E/M

Notes: Acute onset neurological deficit in ED. Comprehensive neurologic examination performed. CT head obtained. Neurology consulted for stroke code activation. Documentation supports high-complexity E/M level.
