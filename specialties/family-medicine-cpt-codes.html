📄 Documentation Requirements

  • ✓ Complete HPI, ROS, physical exam, and MDM for every E/M encounter
  • ✓ Age-appropriate screening documentation for preventive visits
  • ✓ Immunization records with vaccine name, lot number, manufacturer, and site administered
  • ✓ Developmental milestone assessments for pediatric preventive visits
  • ✓ Risk factor counseling and behavior change documentation for preventive services
  • ✓ Documentation of shared decision-making for treatment plan changes
  • ✓ Chronic disease management notes including medication reconciliation
  • ✓ Functional status assessment for disability examinations
  • ✓ Preventive visit checklist completion for age- and gender-appropriate screenings
  • ✓ Documentation of time spent when using total time for E/M level selection

⏱️ Global Surgery Period

Global Period Rules:

0 days for most E/M services and office-based procedures. 10 days for minor procedures (e.g., skin biopsy, joint injection, lesion removal). 90 days for major surgical procedures referred out. Post-op visits (99024) are bundled into the global surgical package and should not be billed separately.

💡 Medical Necessity Tips

  • ✓ Always document the reason for the visit — preventive, problem-oriented, or both
  • ✓ For preventive visits, complete age- and gender-appropriate screening checklist
  • ✓ Document immunization status and any missed or due vaccines at each visit
  • ✓ Match E/M level to documented complexity — medical decision-making or total time
  • ✓ Document chronic disease management with medication adjustments and treatment goals
  • ✓ Include developmental milestone documentation for pediatric patients
  • ✓ Reference USPSTF guidelines for preventive screening appropriateness
  • ✓ Document shared decision-making and patient education for behavior change

📝 Billing Guidelines

  • ✓ Use modifier 25 when billing E/M on same day as immunization or procedure
  • ✓ Bill preventive visit (99395-99397) separately from problem-oriented E/M with modifier 25
  • ✓ Report immunization administration (90471) for each vaccine given plus supply codes
  • ✓ Ensure CCM (99490) is billed once per calendar month with qualifying chronic conditions
  • ✓ Use appropriate place of service codes — POS 11 (office), POS 22 (outpatient hospital)
  • ✓ Verify vaccine supply codes (90476-90474) match actual vaccines administered
  • ✓ Bill well-child visits with appropriate age-specific CPT codes (99394 for under 1 year)
  • ✓ Ensure documentation supports the E/M level selected — do not upcode

📊 Example Claim Scenarios

Annual Wellness Visit with Problem Management

CPT + ICD-10: 99396 (Established patient, 40-64 years) + 99214 + Z00.00 / E11.9 (T2DM)

Modifiers: -25 on 99214 for separate E/M

Notes: Patient presents for annual wellness visit. During visit, blood sugar is elevated requiring medication adjustment. Documentation distinguishes preventive elements from diabetes management. Modifier 25 appended to problem-oriented E/M.

Flu Vaccination with Office Visit

CPT + ICD-10: 99213 (Established patient, low complexity) + 90471 (Vaccine admin) + J1170 (Flu vaccine supply) + I10 (HTN) / Z23

Modifiers: -25 on 99213 for separate E/M

Notes: Patient receives flu vaccine and blood pressure check. Documentation supports separately identifiable E/M for hypertension management. Vaccine administration and supply billed separately.

Pediatric Well-Child Visit

CPT + ICD-10: 99394 (Established patient, under 1 year) + 90471 (Immunization admin) + Z00.121 + Z23

Modifiers: -25 on 90471 if separate from well-child visit

Notes: Well-child visit with age-appropriate immunizations. Developmental milestones documented. Immunization administration billed separately from preventive visit. Vaccine supply codes included for each vaccine given.
