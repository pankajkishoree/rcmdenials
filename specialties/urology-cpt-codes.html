📄 Documentation Requirements

  • ✓ Cystoscopy: indication, findings (normal vs. abnormal), and any biopsies or treatments performed
  • ✓ Prostate procedure: indication, prostate size, pathology findings, and post-op assessment
  • ✓ Kidney stone treatment: stone size, location, number of stones, and treatment method
  • ✓ Ureteral stent placement: indication, laterality, stent type, and follow-up plan
  • ✓ Biopsy documentation: site, technique, number of specimens, and pathology ordered
  • ✓ Voiding diary or urodynamic study results when applicable
  • ✓ Pre-operative assessment and informed consent for all surgical procedures
  • ✓ Post-operative follow-up with catheter status and voiding trial results
  • ✓ PSA and DRE results for prostate-related procedures
  • ✓ Urinalysis and culture results for UTI-related procedures

⏱️ Global Surgery Period

Global Period Rules:

0 days for office-based cystoscopy and diagnostic procedures. 10 days for minor urology procedures (cystoscopy with biopsy, urethral dilation). 90 days for major procedures (TURP, nephrectomy, prostatectomy). ESWL typically has 0-day global period.

💡 Medical Necessity Tips

  • ✓ Document specific urinary symptoms (frequency, urgency, hesitancy, retention) with severity
  • ✓ Include prostate size and PSA results for BPH-related procedures
  • ✓ Document stone size, location, and number for lithotripsy justification
  • ✓ Reference AUA guidelines for appropriate use criteria in urologic procedures
  • ✓ Include failed conservative treatment documentation before surgical intervention
  • ✓ Document urine culture results and antibiotic therapy for UTI-related procedures
  • ✓ Include voiding diary and urodynamic results for incontinence procedures
  • ✓ Document imaging results (CT, ultrasound) supporting stone or tumor diagnosis

📝 Billing Guidelines

  • ✓ Bill cystoscopy with highest-level procedure code performed — do not stack diagnostic and treatment codes
  • ✓ Use modifier 59 for separate procedures at different anatomic sites
  • ✓ Append modifier 25 to E/M when billing on same day as urology procedure
  • ✓ Use modifier 50 for bilateral procedures — bilateral ureteral stents
  • ✓ Bill each stage of TURP (52601) as single code — do not bill separately for each lobe
  • ✓ Ensure laterality documented for all renal and ureteral procedures
  • ✓ Bill ESWL with appropriate stone location code — kidney vs. ureter
  • ✓ Verify global period — post-op visits included in global package

📊 Example Claim Scenarios

TURP for BPH

CPT + ICD-10: 52601 (TURP) + N40.1 (BPH with LUTS)

Modifiers: None — single procedure

Notes: Patient with documented BPH, failed 8 months of alpha-blocker therapy. Prostate size 65 grams. Documented urinary retention requiring catheterization. TURP performed with tissue chips sent to pathology.

ESWL for Kidney Stone

CPT + ICD-10: 50590 (ESWL) + N20.0 (Renal calculus) + N39.0 (UTI)

Modifiers: -RT (right kidney)

Notes: Patient with 12mm right renal calculus causing hydronephrosis. Failed conservative management with hydration and pain control. CT scan confirms stone location. ESWL performed with stone fragmentation documented.

Cystoscopy with Biopsy

CPT + ICD-10: 57452 (Cystoscopy with biopsy) + C67.9 (Bladder cancer)

Modifiers: None — single procedure

Notes: Patient with hematuria and suspicious bladder mass on CT. Cystoscopy with biopsy performed. Multiple specimens sent to pathology. Documentation includes lesion description, location, and size.
