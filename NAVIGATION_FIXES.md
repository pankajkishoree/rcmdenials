## Denial Codes Available

### CO Codes (Contractual Obligation)
1. **CO-4** - Procedure code inconsistent with modifier
2. **CO-16** - Claim lacks information or has submission error
3. **CO-22** - This care may be covered by another payer (COB)
4. **CO-27** - Expenses incurred after coverage terminated
5. **CO-29** - Time limit for filing has expired
6. **CO-45** - Charges exceed fee schedule/contracted rate
7. **CO-50** - Non-covered services (not medically necessary)
8. **CO-97** - Payment adjusted - services bundled

### PR Codes (Patient Responsibility)
1. **PR-1** - Deductible amount

## Features of Each Denial Code Page

- **Comprehensive Guide**: What the code means

- **Sample Appeal Letter**: Template for appeals
