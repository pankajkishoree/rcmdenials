CO-181

CO-181 Denial Code: Payment Adjusted Because

Last Updated: June 2026 | Category: Payment Adjustment

📋 What is CO-181 Denial Code?

CO-181 indicates that payment has been adjusted because the procedure code and modifier combination is not recognized for payment.

🔍 Common Causes of CO-181 Denials

❌

Common Causes

  • Invalid CPT/modifier combination
  • Modifier not applicable to the procedure
  • Code not recognized by the payer
  • Billing error in code selection

✅ How to Resolve CO-181 Denials

1

Review the Denial

Carefully review the remittance advice to understand the specific denial reason.

  • Review the CPT/modifier combination for validity
  • Remove or change the inappropriate modifier
  • Use the correct procedure code for the service
  • Resubmit with corrected coding

📞 What to Ask the Payer

Essential Questions for CO-181 Denials:

  1. "What specific documentation is needed to resolve this denial?"
  2. "What is the appeal deadline for this denial?"
  3. "Can this claim be corrected and resubmitted?"
  4. "Is there a retro-authorization or exception process available?"

💡 Prevention Strategies

  • Validate CPT/modifier combinations before submission
  • Use coding reference tools
  • Train staff on valid code combinations
  • Implement edit checks for common errors
