CO-97

CO-97 Denial Code: Payment Adjusted - Services Bundled

Last Updated: June 2026 | Category: Contractual Obligation

📋 What is CO-97 Denial Code?

CO-97 indicates that payment has been adjusted because the service is included in the payment for another service already performed. This is commonly known as "bundling" and is based on National Correct Coding Initiative (NCCI) edits or payer-specific bundling rules.

The payer considers the denied service to be a component of a more comprehensive procedure and will not pay for it separately. This is a contractual adjustment, not necessarily an error.

🔍 Understanding Bundling Rules

📦

NCCI Edits

  • CMS National Correct Coding Initiative
  • Column 1/Column 2 code pairs
  • Mutually exclusive edits
  • Updated quarterly by CMS
  • Applies to Medicare and many commercial payers
🔗

Common Bundling Scenarios

  • E&M with minor procedures
  • Surgical approach with main procedure
  • Anesthesia with surgical services
  • Lab panels vs. individual tests
  • Imaging guidance with procedures
🔧

Modifier Solutions

  • Modifier 59 - Distinct procedural service
  • Modifier 25 - Significant, separately identifiable E&M
  • Modifier 79 - Unrelated procedure during postop
  • XE, XS, XP, XU - Subset of modifier 59
  • Modifier 91 - Repeat clinical diagnostic lab test
⚖️

Payer-Specific Rules

  • Commercial payer bundling edits
  • State Medicaid bundling policies
  • Facility vs. professional bundling
  • Global surgical package rules
  • Comprehensive vs. component codes

✅ How to Resolve CO-97 Denials

1

Verify the Bundling Edit

Check NCCI edits or payer-specific bundling rules to confirm the codes are legitimately bundled.

  • Review CMS NCCI edit tables
  • Check payer's coding policies
  • Verify effective date of edit
  • Determine if modifier allowed
💡 Resource: Access NCCI edits at CMS NCCI website
2

Review Medical Documentation

Examine the medical record to determine if the services were truly distinct and separately identifiable.

Key Questions:
  • Were the services performed at different anatomical sites?
  • Were they performed at different patient encounters?
  • Was the E&M service significant and separately identifiable?
  • Is there clear documentation supporting separate services?
3

Determine if Modifier is Appropriate

If services were distinct, identify the correct modifier to bypass the bundling edit.

Modifier Selection Guide:
  • Modifier 25: Use when E&M service is significant and separately identifiable from a procedure on the same day
  • Modifier 59: Use when procedures are distinct (different session, different site, separate incision)
  • XE, XS, XP, XU: More specific alternatives to 59 (separate encounter, structure, practitioner, unusual service)
  • Modifier 79: Use for unrelated procedure during postoperative period
4

Resubmit with Appropriate Modifier

If documentation supports separate services, resubmit the claim with the correct modifier.

⚠️ Important: Only use modifiers when documentation clearly supports distinct services. Inappropriate modifier use can be considered fraud.
5

File Appeal with Documentation

If the claim was correctly coded with modifiers but still denied, file an appeal with supporting medical records.

📞 What to Ask the Payer

Essential Questions for CO-97 Denials:

  1. "Which specific codes are being bundled together?"
  2. "What is the basis for this bundling edit - NCCI or payer-specific policy?"
  3. "Is there a modifier that would allow separate payment for these services?"
  4. "What documentation would support unbundling these services?"
  5. "Does your policy allow modifier 59 or the X{EPSU} modifiers?"
  6. "Can you provide a copy of your bundling policy for these codes?"
  7. "If I resubmit with a modifier, should I use frequency code 7?"
  8. "Is there an appeal process if documentation supports separate services?"

💡 Prevention Strategies

  • Use NCCI Edit Software: Implement claim scrubbing tools that check for bundling edits before submission
  • Stay Updated on Edits: Review quarterly NCCI updates and payer policy changes
  • Proper Modifier Usage: Train coders on appropriate modifier application
  • Clear Documentation: Ensure providers document distinct nature of separate services
  • Pre-Submission Review: Check high-risk code combinations before claim submission
  • Payer-Specific Rules: Maintain database of payer-specific bundling policies
  • Coder Education: Regular training on bundling rules and NCCI edits

📊 Common Bundling Examples

E&M with Procedure

Scenario: Office visit (99213) with laceration repair (12002)

Solution: Add modifier 25 to E&M if significant and separately identifiable

Surgical Approach

Scenario: Laparoscopy bundled with laparoscopic cholecystectomy

Solution: Typically bundled - no separate payment

Imaging Guidance

Scenario: Ultrasound guidance with biopsy

Solution: Check if guidance is separately reportable per CPT

Multiple Procedures

Scenario: Two procedures at different anatomical sites

Solution: Use modifier 59 or X{EPSU} modifiers

⚖️ Appeal Guidance

If you believe this denial was made in error, you have the right to appeal. Follow these steps:

  1. Review the denial thoroughly — Understand exactly why the claim was denied.
  2. Gather supporting evidence — Collect all medical records and billing documentation.
  3. Write a detailed appeal letter — Address each denial reason specifically.
  4. Submit within the deadline — Most payers require appeals within 30-180 days.
  5. Follow up regularly — Track your appeal status.

Essential Questions for Your Appeal:

  • "What specific documentation is needed to resolve this denial?"
  • "What is the appeal deadline for this denial?"
  • "Can this claim be corrected and resubmitted?"
  • "Is there a peer-to-peer review option available?"
