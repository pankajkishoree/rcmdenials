CO-170

CO-170 Denial Code: Payment Adjusted When Performed/Billed by This Type of Provider

Last Updated: June 2026 | Category: Contractual Obligation

📋 What is CO-170 Denial Code?

CO-170 is a Contractual Obligation (CO) denial code indicating that payment is adjusted when the service was performed or billed by a type of provider not qualifying for separate payment under the payer's policy. This denial occurs when a provider's specialty, credential type, or practice setting does not meet the payer's requirements for reimbursement of the specific service billed.

The payer restricts certain services to specific provider types or specialties. When a service is submitted by a provider who falls outside the payer's approved provider type for that service, the claim is adjusted or denied accordingly.

🔍 Root Causes of CO-170 Denials

🏥

Provider Type Mismatch

  • Service billed by a provider type not eligible for separate reimbursement
  • Rendering provider's NPI associated with excluded specialty
  • Practice type does not qualify under payer policy
  • Non-qualifying facility or site of service
  • Group practice billing under incorrect provider classification
⚠️

Specialty Restrictions

  • Payer policy restricts certain services to specific provider specialties
  • Service requires a specialist but was billed by a generalist
  • Provider taxonomy code does not match allowed specialties
  • Payer-specific specialty restrictions not followed
  • Credentialing gaps for the provider's specialty
🔄

Incidental Services

  • Service considered incidental to a primary procedure when performed by this provider type
  • Payer bundles incidental services under this provider type
  • Service deemed integral to another billable service
  • Provider type subject to automatic bundling rules
  • Incidental service cannot be separately reimbursed from this provider
📋

Scope of Practice Issues

  • Service falls outside the payer's recognized scope for this provider type
  • State licensure does not authorize this service for the provider type
  • Payer does not recognize this provider type for the service category
  • Scope restrictions vary by payer and plan type
  • Service requires physician supervision not documented

✅ Step-by-Step Resolution Process

1

Review the Denial Details

Examine the ERA/EOB to identify which service and provider type triggered the CO-170 denial. Note the specific service code, rendering provider NPI, and any remark codes indicating the provider type restriction.

💡 Pro Tip: Check the 2300 or 2400 loop of the ERA for provider information and the specific service line affected by the denial.
2

Check Payer Policy for Provider-Type-Specific Billing Rules

Review the payer's provider manual or billing guidelines to understand which provider types are eligible to bill for the denied service. Identify any specialty-specific restrictions or requirements.

3

Verify Rendering Provider's Credential Type and Taxonomy Code

Confirm the rendering provider's credential type, taxonomy code, and specialty classification. Ensure the provider is credentialed and enrolled with the payer under the appropriate specialty for this service.

4

Determine if Service Can Be Reassigned or Rebilled

Evaluate whether the service can be reassigned to a qualifying provider or rebilled under a different provider type. Consider whether a different rendering provider or supervising physician can be used.

⚠️ Important: Ensure any reassignment or provider change complies with payer policies and state regulations. Fraudulent provider changes can result in severe penalties.
5

Submit Corrected Claim

If provider information was inaccurate, submit a corrected claim with the appropriate provider details. Use frequency code 7 (replacement claim) and include the original claim number.

6

File Appeal with Documentation

If the denial is incorrect, file a formal appeal with supporting documentation. Include the provider's credentials, state licensure, and any policy references that support the appropriateness of billing under this provider type.

📞 What to Ask the Payer Representative

Essential Questions for CO-170 Denials:

  1. "Which specific provider type restriction applies to this service?"
  2. "What provider types does the payer allow to bill for this service?"
  3. "Is there a way to rebill under a qualifying provider?"
  4. "What is the appeal deadline for this denial?"
  5. "Can you provide the specific policy section referencing this restriction?"

💡 Prevention Strategies

Reduce CO-170 Denials with These Best Practices:

  • Maintain Provider Credentialing: Ensure all providers are credentialed and enrolled with payers under the correct specialty and taxonomy code
  • Verify Provider Eligibility: Before billing, verify that the provider type is eligible for the specific service with the target payer
  • Review Payer Policies: Stay informed about payer-specific provider type restrictions and specialty requirements
  • Use Provider Validation Tools: Implement software that validates provider type against service codes before submission
  • Document Provider Credentials: Maintain organized records of all provider credentials, licensure, and enrollment status
  • Train Staff on Provider Types: Ensure billing staff understand provider type classifications and their impact on reimbursement
  • Audit Provider Information: Conduct regular audits to verify provider information is accurate and current
  • Check State Requirements: Verify that the provider's state licensure authorizes the service being billed
  • Use Supervision Agreements: For services requiring supervision, ensure proper agreements are documented and on file
  • Monitor Denial Trends: Track CO-170 denials by provider type and service to identify patterns and address root causes
